# Josh Enterprises - Angular Migration

This folder contains the migrated Angular application.

## Prerequisites
- Node.js (Installed)
- Angular CLI (Installed)

## Setup
1. Open this folder in your terminal:
   ```bash
   cd joshenter-angular
   ```
2. Install dependencies (since we skipped install for speed):
   ```bash
   npm install
   ```

## Running the App
1. Start the development server:
   ```bash
   ng serve -o
   ```
2. The application will open at `http://localhost:4200`.

## Notes
- All images are in the `public/` directory.
- Components are located in `src/app/components/`.
- Global styles are in `src/styles.css`.
